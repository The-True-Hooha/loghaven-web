## Install script (PowerShell)

```powershell
irm https://install.loghaven.dev/windows | iex
```

Installs `loghaven.exe` to `%USERPROFILE%\.loghaven\bin` and adds it to your `PATH`.

## Cargo

```powershell
cargo install loghaven
```

## Verify

```powershell
loghaven --version
```

## Run as a Windows service

```powershell
loghaven service install
Start-Service loghaven
```

The daemon communicates via a named pipe (`\\.\pipe\loghaven`) instead of a Unix socket.

## Config location

On Windows, the config lives at `%USERPROFILE%\.loghaven\config.toml`.
