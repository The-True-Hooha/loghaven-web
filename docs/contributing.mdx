LogHaven is built in public under the MIT license. Contributions are welcome.

## Getting started

```bash
git clone https://github.com/The-True-Hooha/loghaven
cd loghaven
cargo build
cargo test
```

## Areas to contribute

- **Chain adapters** — new chain support (EVM L2s, Cosmos, etc.)
- **Storage backends** — GCS, Azure Blob, IPFS
- **Query engine** — filtering, aggregations, search
- **CLI / TUI** — UX improvements
- **Documentation** — this site

## Opening a PR

1. Fork the repo
2. Create a branch: `git checkout -b feat/your-feature`
3. Make your changes with tests
4. Open a PR against `main`

Please open an issue first for large changes so we can align on direction.

## Code of conduct

Be direct, be helpful, be honest. No jargon, no hype.
