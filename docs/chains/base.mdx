---
title: "Base"
description: "Configure the Base adapter"
---

Base uses the same EVM adapter as Ethereum. See the [Ethereum adapter](/chains/ethereum) page for the full config reference.

## Config

```toml
[adapters.base]
enabled  = true
rpc      = "https://mainnet.base.org"
chain_id = 8453
```

Public RPC alternatives:

| Provider | URL |
|---|---|
| Base public | `https://mainnet.base.org` |
| Alchemy | `https://base-mainnet.g.alchemy.com/v2/YOUR_KEY` |
| Infura | `https://base-mainnet.infura.io/v3/YOUR_KEY` |
